# ⚔️ 검 강화 게임

검을 계속 강화하는 노가다 클리커 게임입니다.

## 🎮 게임 방식

- **강화하기**: 100 골드를 소비하여 검을 강화할 수 있습니다.
- **성공 확률**: 강화 레벨이 높아질수록 성공 확률이 낮아집니다.
- **보상**: 성공 시 경험치 10을 얻고, 실패 시 경험치 1을 얻습니다.
- **상점**: 골드를 더 얻을 수 있습니다. (테스트/카주용)
- **통계**: 현재 진행도와 성공률을 확인할 수 있습니다.

## 🚀 설치 및 실행

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 빌드
npm run build
```

## 📊 강화 확률 시스템

| 레벨 범위 | 성공 확률 |
|----------|----------|
| 1-4 | 100% |
| 5-9 | 90% |
| 10-14 | 70% |
| 15-19 | 50% |
| 20-24 | 30% |
| 25-29 | 15% |
| 30+ | ~1% |

## 💾 데이터 저장

모든 진행도는 브라우저의 localStorage에 자동으로 저장됩니다.
